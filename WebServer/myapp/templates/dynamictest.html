Welcome to Anomalyze!

About: 
Anomalyze leverages AI to create an anomaly detection model to analyze live network traffic for 
suspicous activity with real time visualizations.

How It All Works: 
By using your devices network interface, IPv4 TCP packets are captured and processed by the Anomalyze 
trained model which then yeild the results. This model is trained to detect buffer overflow and SYN 
flood attacks in network traffic with X amt of accuracy.

How to Use: 
In order to use Anomalyze first you must specify a destination IP address that you will faciliate 
communication with. Once the "Start Session" button is pressed the packet caputre will begin and the 
visualizations will be updated with the live data. When pressing the save session button, a JSON file 
will be saved holding the information of the packet capture session. Pressing the stop session button 
will discontinue the packet capture process between devices, but note that does not stop communication 
between these devices.

Enjoy your packet scanning experience!
